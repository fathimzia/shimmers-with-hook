# Shimmers with Hook 🧶

A beautiful e-commerce website for handmade crochet creations and magical slime products.

## Features
- 🛒 Shopping cart with localStorage persistence
- ✓ Form validation with email checking
- 🔔 Toast notifications
- 📱 Fully responsive design
- 🎨 Beautiful gradient UI with hover effects

- Flowers (₹199)
